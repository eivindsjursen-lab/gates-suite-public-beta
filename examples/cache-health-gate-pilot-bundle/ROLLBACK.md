# Rollback (Public Beta Pilot)

Rollback should be fast and low-risk.

## Remove the pilot completely

Delete:

- `.github/workflows/cache-alpha.yml`
- `.github/actions/cache-health-gate/`

This removes:

- the local action bundle
- the beta workflow
- any PR comments generated by prior runs remain as history, but no new runs will happen

## Keep workflow, remove gate only

If you want to keep the workflow for later and only stop the gate:

- remove or comment out the `cache-gate` job
- or remove the `Run Cache Health Gate` step

## No application/runtime impact

The pilot bundle only affects CI workflow behavior.
It does not change application runtime code.
